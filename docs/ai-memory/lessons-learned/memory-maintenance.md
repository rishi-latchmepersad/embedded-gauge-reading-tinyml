# Memory maintenance

Date: 2026-07-22  
Status: current  
Scope: project documentation and AI-agent handoff

- Keep current facts short and separate from chronology.
- Put one decision or incident in one dated note.
- Link to evidence instead of copying long logs or entire experiment reports.
- Mark stale notes `superseded` and link the replacement; do not silently edit
  historical results.
- Use `archive/` for old snapshots so the active folders remain useful at a
  glance.
